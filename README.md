# Bakery-System
The system handles the complete order lifecycle — from customer browsing and checkout to kitchen processing, inventory management, and real-time WhatsApp notifications — with zero manual intervention on the operational side.

## Stack

- **Frontend**: Lovable (React)
- **Database & Auth**: Supabase (PostgreSQL)
- **Automation**: Make.com
- **Notifications**: WhatsApp

## Key Features

- Customer storefront with cart, custom cake requests, and simulated checkout
- Kitchen dashboard with real-time order status tracking
- Admin portal with sales KPIs and inventory monitoring
- Automated WhatsApp alerts for new orders, status changes, and low stock
- Google Calendar integration for delivery scheduling
- Inventory deduction on order processing with safety threshold alerts
